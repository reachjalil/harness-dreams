# Harness Dreams Brand Theme Variants

Brand-aligned image-generated marketing assets using the Harness Dreams logo, app UI theme, and website hero palette.

## Brand Cues Used

- Moon-orbit Harness Dreams logo
- Ink navy: `#0d1b2a`
- Dusk blue: `#4a6cf7`
- Pale teal: `#a7dad3`
- Deep teal: `#3aab9c`
- Violet: `#7a5af0`
- Light app surface, lavender sidebar, purple action button, and blue/teal/violet dream-score rings

## Files

- `source-generated/` - raw image-generation outputs using the logo/theme references.
- `final/` - production copies with the exact Harness Dreams mark composited over the generated logo area.
- `print-300dpi/` - 2x resized PNG exports with 300 DPI metadata.
- `presentation/04-themed-widescreen-exact-logo-4k.png` - 3840x2160 widescreen slide export.
- `assets/brand-mark-navy.png` and `assets/brand-mark-white.png` - transparent raster marks drawn from the site logo geometry.

## Production Picks

- `final/01-themed-overview-exact-logo.png` - broad overview handout with Cloud Sync chip.
- `final/02-app-theme-cycle-ready-exact-logo.png` - closest to the desktop app screenshot and product UI.
- `final/03-branded-cloud-sync-exact-logo.png` - Cloud Sync pricing card.
- `presentation/04-themed-widescreen-exact-logo-4k.png` - widescreen presentation slide.

## Notes

- The main poster text and product scenes are generated inside the artwork.
- The exact logo mark is composited afterward so the brand mark stays consistent.
- Review small in-app mockup text before commercial printing; generated UI microcopy can contain artifacts.
