# Harness Dreams Cloud Sync Print

Print-oriented marketing asset for the optional iPhone and Apple Watch Cloud Sync upgrade.

Files:

- `source/cloud-sync-artwork.png` - AI-generated artwork background.
- `final/cloud-sync-poster.png` - final 4:5 poster with deterministic typography.
- `generate-poster.py` - regenerates the final poster from the source artwork.

Regenerate:

```bash
python3 marketing/cloud-sync-print/generate-poster.py
```

The source artwork was generated with the built-in image generation tool. The final copy is drawn locally so pricing and product claims stay exact.
